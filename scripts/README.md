# Scripts

Scripts automate documentation, validation, release, and conformance workflows.

## Planned Scripts

| Script | Purpose |
|---|---|
| `validate-openapi` | Validate OpenAPI 3.1 contract |
| `validate-schemas` | Validate JSON Schema documents and examples |
| `check-links` | Verify Markdown links |
| `lint-markdown` | Enforce Markdown style |
| `build-docs` | Generate developer portal pages |
| `generate-pdfs` | Generate PDF versions from canonical Markdown |
| `run-conformance` | Execute protocol conformance suite |

## Existing Utility

The canonical docs package includes [`docs/aifp/md2pdf.py`](../docs/aifp/md2pdf.py), used to render Markdown docs to PDF with browser print output.

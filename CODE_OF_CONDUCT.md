# Code of Conduct

AiFinPay Paywall Protocol is built for a professional, standards-oriented community.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while staying precise.
- Keep technical discussions evidence-based.
- Give actionable feedback.
- Respect security disclosure boundaries.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private security reports before remediation.
- Spam, trolling, or repeated off-topic disruption.

## Enforcement

Maintainers may moderate comments, close issues, restrict participation, or escalate to organization administrators when needed.

## Reporting

For conduct issues, contact maintainers privately. For security vulnerabilities, use `SECURITY.md` instead.

# Assets

Brand, diagram, and visual assets for AiFinPay Paywall Protocol.

## Structure

| Path | Purpose |
|---|---|
| `brand/` | Logo marks, color palette, typography guidance |
| `diagrams/` | Exported architecture diagrams |
| `screenshots/` | Developer portal screenshots |

## Brand System

| Token | Value | Use |
|---|---|---|
| Midnight | `#0B1020` | Primary text and deep surfaces |
| Protocol Blue | `#2F5FD6` | Primary brand/action color |
| Settlement Teal | `#14B8A6` | Success, settlement, receipts |
| Signal Amber | `#F59E0B` | Warnings, draft status |
| Cloud | `#F4F6FB` | Soft backgrounds |

See [`docs/aifp/13-Branding-Guidelines.md`](../docs/aifp/13-Branding-Guidelines.md) for canonical brand guidance.

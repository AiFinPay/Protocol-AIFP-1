# AiFinPay Brand Assets

## Logo

- Primary mark: [`aifinpay-paywall-mark.svg`](aifinpay-paywall-mark.svg)
- Use on light or dark backgrounds with adequate spacing.

## Palette

| Token | Hex | Use |
|---|---|---|
| Midnight | `#0B1020` | Deep surfaces, primary text |
| Protocol Blue | `#2F5FD6` | Primary action, links, protocol highlights |
| Settlement Teal | `#14B8A6` | Receipts, settlement, success states |
| Signal Amber | `#F59E0B` | Draft status, warnings, migration notes |
| Cloud | `#F4F6FB` | Soft documentation backgrounds |

# AiFinPay Brand Palette

| Token | Hex | Usage |
|---|---|---|
| Midnight | `#0B1020` | Primary text, dark hero surfaces |
| Protocol Blue | `#2F5FD6` | Primary actions, protocol marks |
| Settlement Teal | `#14B8A6` | Receipts, settlement, success |
| Signal Amber | `#F59E0B` | Draft status, warnings |
| Cloud | `#F4F6FB` | Soft backgrounds |
| Slate | `#334155` | Secondary text |

Typography should prioritize clear technical reading: Inter, Helvetica Neue, Arial, or system sans-serif for prose; JetBrains Mono, SF Mono, Consolas, or monospace for code.
